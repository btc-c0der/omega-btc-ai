
### Command Line Arguments

| Argument | Description | Default |
|----------|-------------|---------|
| `--refresh_interval` | Refresh interval in seconds | 1.0 |
| `--matrix_interval` | Show matrix rain effect every N frames | 30 |
| `--demo` | Run in demo mode with mock data | - |

### Examples

