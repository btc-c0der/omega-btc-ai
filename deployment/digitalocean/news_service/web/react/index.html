About the Fear & Greed Index API

The Crypto Fear & Greed Index analyzes emotions and sentiments from different sources and produces a single number: the Fear & Greed Index for Bitcoin and other large cryptocurrencies.

A value of 0 means "Extreme Fear" while a value of 100 represents "Extreme Greed".
